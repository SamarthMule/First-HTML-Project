How HTML works ?

HTML Document => Browser => Rendered Page
Basic HTML page

        DOCTYPE HTML     - Specifies this an htma 5 doc
        HTML                       -  Root of HTML page
            head                    - Contains page metadata
                tital /tital        - Contain tital
            /head
            body                    - main body which is rendered by Browser
                h1 /h1             - Heading tag
                p                       - paragraph tag

                /p
            /body
        /HTML

HTML element = Start tag + Content + End tag
HTML is case Insensitive language
