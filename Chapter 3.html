HTML element = Start tag + Content + End tag
Some Imp Tages
  • Heading Tag - [h1,h2,h3,...,h6]
  • paragraph Tag - [p]
  • Anchor Tag - [a href="(Any Link,mail,tel,etc)"]
  • Image Tag - [img src = "(Address,link) width="(width of image)"]
  • Video Tag - [Video scr = "(Any Link of Video)"](Message like error)[/Video]
  • Bold Tag - [b]
  • Italic Tag - [i]
  • Undreline Tag - [u]
  • Line Break - [br]
  • Horizontal Rular/line - [hr]
  • Subscript Tag - [sub]
  • Superscript Tag - [sup]
  • As it is - [pre]
  • Bit larger - [big]
  • Bit smaller - [small]
  • font - [font color="" size="" face=""]
  • list - [li] & [ul type]
  • Inline Container - [span]
  • Container(block level element) - [div]
IMP Page Layout
  • Header for nav tags- [Header] & [nav]
  • main opening - [main]
  • Page section - [section]
  • Self Contained Content - [article]
  • Content Aside from content(eg. Ad's,etc) - [aside]
  • Footer for nav tags - [footer]
